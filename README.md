# Airline-Management-System
Airline Management System is a console-based C++ application that simulates the operations of an airline. It allows administrators to manage flights and users to search, book, and view flight details. All data is dynamically stored in text files, ensuring persistence across sessions.


Features
---------------------------------------------------
Admin Portal
---------------------------------------------------
Login with username and password.

Add, delete, or update flight details.

Update total seats, fare, and flight status.

View all flights or fully booked flights.

Search flights by Flight ID.

View all user bookings.

------------------------------------------------------
User Portal
------------------------------------------------------

User registration (Sign Up) and login.

Search for flights by Flight ID.

Book tickets with seat availability tracking.

View personal bookings.

View ALL Available Flights.

--------------------------------------------------------
Core Functionalities
--------------------------------------------------------

Dynamic memory allocation for storing flight data.

File-based persistent storage for flights, bookings, and user/admin credentials.

Automatic Booking ID generation.

Handles seat availability and prevents overbooking.
